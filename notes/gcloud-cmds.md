To check accounts  
```
$ gcloud auth list
```
to get the details
```
$ gcloud info
$ gcloud config list
```
to list components, install, update
```
$ gcloud components list
$ gcloud components install <name>
$ gcloud components update
```

Use interactive command line to get auto suggestions 
```
$ gcloud beta interactive
```
